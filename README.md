## Description

Simple server build with Nest.JS framework that offers, endpoints to get the weather by city and convert rate to another rates.

* This server is the main server for my app Simply Util for iOS that will replace Simply Convert and Simply Forecast.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
